NumPy and Linear Systems

Summary

NumPy (short for numerical Python) is the core component for just about all technical computing done with Python. In this lesson, we examine its basic components with a focus on arrays and linear systems.

Resources

The NumPy project provides a lot of documentation and support, including this introductory tutorial.

Evidence of Student Learning

  • Students will install (if needed) and import NumPy.
  • Students will construct NumPy arrays to represent vector quantities.
  • Students will construct NumPy arrays to represent matrix quantities.
  • Students will use NumPy to solve linear systems of the form .
  • Students will reflect on their learning by completing their daily log.

Learning Plan

Before Lecture

  • Review linear systems.
  • Review least squares

During Lecture

  1. Review basic array operations.
  2. Set up a linear system using NumPy arrays for a small heat-conduction problem. We will use this open-source example code
  3. Discuss (and show!) the challenges of systems when they become very large.
