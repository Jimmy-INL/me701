Sources of Numerical Error

Summary

Computers most often work with data encoded in a binary (1’s and 0’s) format. We humans tend to think in decimal. To go between the two formats on a machine with finite memory requires we make approximations all the time, even if we don’t notice. We’ll cover binary and decimal representation with a focus on numerical applications. However, the core ideas are important to any low-level signal processing and data handling that we do later on.
