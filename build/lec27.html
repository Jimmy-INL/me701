GUI Layouts

Summary

Students will construct GUI apps with multiple widgets using available layout and other organizational widgets.

Resources

https://doc.qt.io/qt-5/widget-classes.html#layout-classes

https://zetcode.com/gui/pyqt5/layout/

Evidence of Student Learning

  • Students will construct a basic pin pad with auxiliary signals.
  • Students will connect the pin pad to a simple game of “guess the number.”

Learning Plan

Before Lecture

  • Skim documentation for the available organization widgets.
  • Review positioning of Widgets

During Lecture

  1. Clone the sample code for this lesson.
  2. Explore use of QWidget.move for absolute positioning.
  3. Explore use of “box”, “grid”, and “stacked” layouts.
  4. Explore use of “tabs”
