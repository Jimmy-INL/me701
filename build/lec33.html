PyQt and Custom Widgets

Summary

Students will learn how to interactively paint on a QPixmap and to use the resulting data to infer information about a background canvas image.

Resources

Martin Fitzpatrick, Create GUI Applications with Python & Qt5.

Alan Moore, Mastering GUI programming with Python.
