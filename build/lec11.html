The Structure of Python Programs

Summary

The basic structure of Python code is discussed using conditional statements (i.e., if), loops, and functions to illustrate.

Resources

codewars is an online resource that gamifies the code-building experience. The site emphasizes test-driven development.

Evidence of Student Learning

  • Students will explain what a Python program does (or doesn’t) do.
  • Students will produce code to solve simple problems.
  • Students will reflect on their learning by completing their daily log.

Learning Plan

Before Lecture

  • A laptop with Anaconda.
  • Review lectures 13 and 14 of Python for Engineers.
  • Sign up for codewars.

During Lecture

  1. Clone repository with sample programs that illustrate fundamental structures.
  2. Provide 2 to 3 in-class exercises to build on demonstration programs.
  3. Quick walk through of codewars.
